What i do

Deep Learning & Computer Vision

With expertise in machine learning and deep learning, I build and optimize computer vision models for object detection, segmentation, and tracking. I work with frameworks like PyTorch, TensorFlow, and OpenCV, ensuring high-performance solutions.

3D Vision & Point Cloud Processing

I specialize in 3D vision, LiDAR data processing, and point cloud analysis using PCL, Open3D, and custom deep learning models. My experience includes 3D reconstruction, object recognition, and non-rigid object tracking.

Model Development & Deployment

I integrate AI-driven vision systems into robotics and automation. I have hands-on experience with camera calibration, real-time object tracking, and synthetic data generation for ML models.

AI & Robotics Applications

I design and train state-of-the-art AI models, fine-tuning them for real-world applications. I also handle model optimization, deployment, and integration into scalable systems using Docker, TensorRT, and ONNX.

Research & Innovation

Passionate about AI research, I explore self-supervised learning, generative models, and novel deep learning architectures to push the boundaries of computer vision technology.

3D Pose Estimation

This project estimates the 6D pose (position and orientation) of a dominant "floating" box in an RGB-D scene. It uses Open3D for point cloud processing, DBSCAN for clustering, and RANSAC for plane segmentation to identify the top surface of the box and subsequently calculate its pose.

3D Object Detection

This project implements a 3D object detection pipeline using a fusion of image and point cloud data. The model is trained to predict 3D bounding boxes from RGB images and corresponding LiDAR point clouds.

2D Depth Estimation

This project shows the integration of Yolov3 with Midas for depth estimation. The model is trained to predict depth information of 2d RGB images.
